BioRemPP v1.0.0-beta - Initial Public Release

Bioremediation Potential Profile Web Service
Official release for organization repository.

RELEASE HIGHLIGHTS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SCIENTIFIC CAPABILITIES
  • 56 analytical use cases across 8 specialized modules
  • Integration with 4 curated databases (BioRemPP, KEGG, HADEG, toxCSM)
  • 7 regulatory framework classifications
  • Comprehensive toxicological risk assessment

VISUALIZATIONS
  • 19 chart types (heatmaps, networks, sunbursts, dendrograms, etc.)
  • Interactive Plotly-based visualizations
  • Export to PNG, SVG, JPEG formats
  • Responsive graphics

ARCHITECTURE
  • Clean Architecture (4-layer design)
  • Strategy Pattern for plot generation
  • Factory Pattern for dynamic panels
  • YAML-based configuration system
  • Multi-layer caching (Memory, DataFrame, Graph)

DEPLOYMENT
  • Docker Compose orchestration
  • Development and production environments
  • Gunicorn + gevent workers
  • Health checks and monitoring
  • Nginx reverse proxy support

DOCUMENTATION
  • README.md - Complete setup and usage guide
  • NumPy-style docstrings throughout
  • Comprehensive user guide
  • API reference documentation
  • Scientific methodology docs

DATA POLICY
  • No user accounts required
  • Session-based processing
  • No persistent data storage
  • Privacy-first design
