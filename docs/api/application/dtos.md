# Data Transfer Objects (DTOs)

Application Layer - DTOs Package.

This package contains Data Transfer Objects (DTOs) used for transferring data between
different layers of the application while maintaining loose coupling.

## Modules

- **[Upload Result DTO](dtos/upload_result_dto.md)**: Result of file upload operations
- **[Validation Result DTO](dtos/validation_result_dto.md)**: Result of data validation operations
- **[Merged Data DTO](dtos/merged_data_dto.md)**: Merged data from multiple databases
- **[Processing Progress DTO](dtos/processing_progress_dto.md)**: Progress tracking information
