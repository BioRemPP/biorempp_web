# Core Handlers

Core Operations Module.

This module contains the core application operations and handlers that orchestrate business
workflows. Core operations handle upload processing, data parsing, merge operations, and
result export functionalities.

## Modules

- **[Upload Handler](core/upload_handler.md)**: Handles file upload validation and processing
- **[Data Processor](core/data_processor.md)**: Orchestrates data processing and merge operations
- **[Sample Parser](core/sample_parser.md)**: Parses input data into domain entities
- **[Result Exporter](core/result_exporter.md)**: Exports analysis results in various formats
