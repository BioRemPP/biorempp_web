# Data Tables

Presentation Layer - Data Tables Package.

This package contains components for rendering interactive data tables for each database
in the BioRemPP application. Each table provides filtering, sorting, and export capabilities
for exploring the merged datasets.

## Modules

- **[BioRemPP Table](data_tables/biorempp_table.md)**: Interactive table for BioRemPP database data
- **[HADEG Table](data_tables/hadeg_table.md)**: Interactive table for HADEG database data
- **[KEGG Table](data_tables/kegg_table.md)**: Interactive table for KEGG database data
- **[ToxCSM Table](data_tables/toxcsm_table.md)**: Interactive table for ToxCSM database data

## Features

All data tables include:

- **Filtering**: Column-based filtering capabilities
- **Sorting**: Multi-column sorting support
- **Pagination**: Efficient handling of large datasets
- **Export**: Download data in CSV, Excel, or JSON formats
- **Responsive Design**: Adapts to different screen sizes
