## Overview of User Guide Content

### Results Page Structure

The **Results Page** is your analytical workspace after successful data upload. It organizes:

- **8 analytical modules** containing **56 use cases** total
- **Database integration tables** (BioRemPP, KEGG, HADEG, ToxCSM)
- **On-demand execution** for visualizations and data tables
- **Session-based architecture** (4-hour lifetime)

**[Learn about Results Page organization →](results-page.md)**

---

### Scientific Interpretation

Results represent **genetic potential** based on KO annotations, not confirmed biological activity. Key principles:

- Distinguish functional capacity from enzymatic activity
- Understand regulatory classifications as informational context
- Recognize toxicity predictions as computational estimates requiring validation
- Avoid common misinterpretations that may lead to invalid conclusions

⚠️ **[Read interpretation guidelines →](interpretation.md)**

---

### Data Export and Reproducibility

Export results for external analysis, scientific publication, or long-term archival:

- **Tabular formats:** CSV, Excel, JSON
- **Visualization formats:** PNG, SVG, JPEG
- **Two export scopes:** Database-level (complete datasets) and use-case-level (specific analyses)
- **Reproducibility requirements:** Document BioRemPP version, analysis date, parameters

**[View export documentation →](downloads.md)**

### Troubleshooting Technical Issues

Resolve common problems with structured guidance:

- **Processing errors:** Validation failures, encoding issues, sample/KO limits
- **Performance issues:** Timeouts, slow response, session management
- **Visualization problems:** Charts not displaying, rendering errors, export failures
- **Download issues:** Unresponsive buttons, corrupt files, format problems

**[Access troubleshooting guide →](troubleshooting.md)**
