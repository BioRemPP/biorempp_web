feat: initial commit - BioRemPP v1.0.0-beta web service

Complete implementation of BioRemPP (Bioremediation Potential Profile) 
web service for functional analysis of microbial communities.

FEATURES:
- 8 specialized analytical modules (56 use cases total)
- Clean Architecture implementation (4-layer design)
- Integration with curated databases (BioRemPP, KEGG, HADEG, toxCSM)
- 7 regulatory framework classifications (IARC, EPA, ATSDR, WFD, PSL, EPC, CONAMA)
- Interactive visualizations (19 chart strategies via Plotly)
- Multi-format data export (CSV, Excel, JSON)
- Docker deployment (development + production)
- Comprehensive documentation (NumPy-style docstrings)

ARCHITECTURE:
- Domain layer: Business logic and plot strategies
- Application layer: Use case orchestration
- Infrastructure layer: Database integration and caching
- Presentation layer: Dash-based UI with 56 interactive panels

DATABASES:
- BioRemPP Database v1.0 (2025)
- KEGG Release 116.0+/12-19, Dec 25
- HADEG: Commit 8f1ff8f (2023)
- toxCSM: v1.0 (2022)

DEPLOYMENT:
- Python 3.11+ with Dash framework
- Docker Compose for containerization
- Gunicorn + gevent for production
- Development and production configurations

DOCUMENTATION:
- Complete README 
- Comprehensive API documentation
- User guide and FAQ
- 56 use case documentation pages
- Scientific methodology documentation

This release represents the complete beta version pending article publication.
